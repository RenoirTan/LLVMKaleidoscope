Expand description

A token in a Kaleidoscope file.

This structural representation of a token contains the possible TokenKind of the token, the token as a string (stored as span), as well as the start and end indices of the token.

    &mut self,
    unit: char,
    index: FileIndex
) -> Result<bool, Error<ErrorKind>>

[src]

Add a character into the token. This method returns a boolean value if no error occurs.

If true is returned, this means that unit is a character that this token cannot accept, so this token is complete. If false is returned, it means that unit has been added to the token’s span and can continue to collect more characters.
